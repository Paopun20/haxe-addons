# Haxe Addons

A curated collection of useful addons, utilities, and experimental modules for the Haxe programming language.

## Installation

```sh
haxelib git haxe-addons https://github.com/Paopun20/haxe-addons
```

Then include it in your project.

=== "Base Haxe"

    ```hxml
    -lib haxe-addons
    ```

=== "Lime | OpenFL"

    ```xml
    <haxelib name="haxe-addons" />
    ```
