# IntTools

Static utility class for integer operations.

```haxe
import haxe.addons.tools.IntTools;
```

## Methods

| Method                                  | Description              |
| --------------------------------------- | ------------------------ |
| `inRange(v:Int, min:Int, max:Int):Bool` | Inclusive range check    |
| `isEven(v:Int):Bool`                    | Check if even            |
| `isOdd(v:Int):Bool`                     | Check if odd             |
| `toFloat(v:Int):Float`                  | Convert to float         |
| `digitCount(v:Int):Int`                 | Number of decimal digits |
| `gcd(a:Int, b:Int):Int`                 | Greatest common divisor  |
| `lcm(a:Int, b:Int):Int`                 | Least common multiple    |
