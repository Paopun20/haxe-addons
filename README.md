# Haxe Addons

> A curated collection of useful addons, utilities, and experimental modules for the Haxe programming language.

## Collections

Utilities and data structures.

- **Deque** — Double-ended queue implementation.

## Math

General-purpose mathematical utilities.

- **Interpolation** — Common interpolation functions.
- **MathTypes** — Shared math-related types.
- **Noise** — Base API for procedural noise generation.

### Noise Algorithms

Procedural noise generators for terrain generation, textures, and simulations.

- **CellularNoise** — Cellular (Voronoi) noise.
- **CurlNoise** — Curl noise for vector fields.
- **DomainWarp** — Domain warping utilities.
- **Fractal** — Fractal noise combiners (FBM, Ridged, etc.).
- **NoiseMap** — Generate and manipulate noise maps.
- **NoiseTypes** — Common noise-related enums and types.
- **OpenSimplex2** — OpenSimplex2 noise implementation.
- **PerlinNoise** — Classic Perlin noise.
- **SimplexNoise** — Simplex noise implementation.
- **ValueNoise** — Value noise implementation.
- **WorleyNoise** — Worley (cellular) noise implementation.

## System

Platform and filesystem helpers.

- **Path** — Path manipulation utilities.

## Tools

Extensions and helper functions for built-in types.

- **ArrayTools** — Array helper methods.
- **FloatTools** — Float utilities.
- **IntTools** — Integer utilities.
- **StringTools** — Additional string helpers.

## Types

Custom primitive and utility types.

- **BitField** — Bit manipulation helper.
- **Int8** — Signed 8-bit integer type.
- **UInt8** — Unsigned 8-bit integer type.

## Useless

Completely impractical classes created for fun.

- **ArrayOfBabel** — An array that probably shouldn't exist.
- **BlackHole** — Consumes everything.
- **QuantumBoolean** — Simultaneously `true` and `false` until observed.
